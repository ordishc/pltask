### Installing

A step by step series of examples that tell you how to get a development env running


```
* Install node v10.15.1 (latest)
* npm version 6.4.1
* npm install
* cd into root folder
* run 'gulp build:dev' and the site should spin up, if not head to localhost:3000

```
